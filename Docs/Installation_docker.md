# Installation en utilisant Sail (Docker)

**Sail** permet de mettre en place un *container Docker* facilement. Cela permet ainsi de faire tourner une application Laravel sans faire toutes les étapes d'installation laborieuses, telles que dans les autres tutos d'installation.

**Remarque** : En *production*, HypAIR tourne justement dans un container Docker, mais fait à la main, sans passer par *Sail*.

Voici un guide pour mettre en place un environnement de développement pour HypAIR, en utilisant **Laravel Sail**.
